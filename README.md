# TFM_PARKINSON
Repositorio con el código desarrollo para TFM del Máster en Visión Artificial. 
*Autor:* Walter Rene Cuenca Guachamin.

# Composicion
Los desarrollos se encuentran distribuidos en directorios que corresponden a los distintos experimentos expuestos en el proyecto. A continuación, se indica el título
de cada uno de ellos:
* *Data*: se describe la base de datos empleada en el proyecto.
* *Procesos Iniciales:* recoge el conjunto de procesos previos a los experimentos. Son necesarios para preparar los datos para las
  etapas posteriores.
* *Experimento I:* imágenes completas con redes convolucionales.
* *Experimento II:* imágenes completas usando modelos opuestos de Redes Convolucionales.
* *Experimento III:* fragmentos de imágenes con Redes Convolucionales.
* *Experimento IV:* fragmentos de imágenes con modelos opuestos con Redes Convolucionales.
* *Experimento V:* imágenes completas con *Vision Transformer (ViT)*.
